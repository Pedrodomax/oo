BIOS
Logo após, quem entra em ação é o BIOS (Basic Input/Output System — Sistema Básico de Entrada/Saída em português), um sistema operacional pré-gravado no chipset que garante a tradução dos códigos de hardware para a tela — sua interface de configuração (Setup Utillity) é azul, sendo facilmente reconhecida por muitos usuários. BIOS também é responsável por carregar a memória RAM, placa de vídeo, teclado, cachê básico e, por fim, possibilitar a inicialização do sistema operacional.
